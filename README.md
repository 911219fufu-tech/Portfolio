# Joyce Fu Portfolio

Live Demo: [joycefuportfolio.vercel.app](https://joycefuportfolio.vercel.app/)

This repository contains my personal portfolio website, built to present my work in frontend development, UX/UI design, and product thinking in one place.

The site highlights selected projects, experience, education, technical skills, and contact information in a clean case-study format designed for quick review by recruiters, hiring managers, and collaborators.

## What This Portfolio Includes

- A curated project section with product, frontend, and fullstack work
- Detailed case study pages with problem framing, features, implementation, and reflection
- An experience section that connects real-world work to workflow thinking and product decisions
- Education, technical skills, and contact pages
- A responsive UI with motion, visual hierarchy, and custom styling

## Featured Work

- **JF BrokerMap**  
  A UX/UI + frontend dashboard for leasing workflows that brings maps, pricing, availability, OP, and notes into one structured interface.

- **ChronoSense**  
  A time-awareness product concept built during FigBuild 2026, focused on attention patterns and intentional workflows.

- **UNLMTD**  
  A fitness recovery app prototype that translates health and training signals into clear decisions.

- **Joystick Journal**  
  A full-stack game review platform built with React, Node.js, and MySQL.

## Tech Stack

- React
- Vite
- React Router
- Tailwind CSS
- Framer Motion

## Design Goals

- Present projects as thoughtful case studies, not just screenshots
- Show both technical execution and product/UX decision-making
- Keep the experience visually polished, responsive, and easy to scan
- Make it easy for recruiters to understand my background quickly

## Local Development

```bash
npm install
npm run dev
```

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Project Structure

```text
src/
  components/   reusable UI components
  data/         portfolio content and project metadata
  pages/        route-level pages
  sections/     page sections such as Projects, Experience, and Skills
  assets/       project visuals and images
```

## Why I Built This

I wanted a portfolio that reflects how I actually work: combining frontend execution, UX/UI thinking, and structured product decisions. Rather than treating design and engineering as separate tracks, this site is meant to show how those strengths come together in the work I build.
